# HR Analytics Dashboard – Power BI

## Overview

This project presents an interactive **HR Analytics Dashboard** built using Power BI to analyze employee data and workforce trends within an organization. The dashboard provides insights into employee distribution, recruitment sources, performance scores, salary patterns, and employee satisfaction.
The goal of this project is to help HR teams understand workforce dynamics and make data-driven decisions related to employee management and organizational performance.

## Dashboard Insights
The dashboard provides several key HR insights including:

* Total number of employees in the organization
* Employee distribution by department
* Gender diversity analysis
* Employee marital status distribution
* Recruitment source analysis
* Performance score distribution
* Employee satisfaction and engagement scores
* Average salary analysis by department and gender
* Employee distribution by state

## Dashboard Features

* KPI cards for quick overview of workforce statistics
* Department-wise employee distribution
* Gender diversity visualization
* Recruitment trend analysis
* Performance evaluation insights
* Salary analysis across departments
* Employee satisfaction and engagement metrics

## Tools & Technologies Used

* Power BI

* Data Cleaning & Data Transformation
